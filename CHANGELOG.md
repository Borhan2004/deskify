## 0.1.0

- Initial release of Deskify core foundation.
- Added `DeskShell` with adaptive navigation (Bottom Bar vs Sidebar) and Glassmorphism support.
- Added `HoverDecorator` for easy hover interactions with smooth animations.
- Added `DeskConstraintBox` for layout management.
- Initial boilerplate for Accelerator (Keyboard Shortcuts) and Right-Click Engine.
- Enhanced Example app with "Top-Notch" designer-grade UI and modern typography.
