# Deskify Example 🚀

This is a premium example of the **Deskify** package, showcasing a modern, high-end Todo/Dashboard application.

## Features Showcased
- **Adaptive Shell**: Resize the window to see the sidebar switch to a bottom bar.
- **Glassmorphism**: Experience the frosted glass effect in the desktop sidebar.
- **Hover Interactions**: Hover over task cards and buttons to see subtle scaling and color shifts.
- **Right-Click Menu**: Right-click on any task card to see the custom context menu.
- **Keyboard Shortcuts**: Press `Ctrl+N` (or `Cmd+N` on Mac) to quickly add a new task.

## Running the Example
From the root directory:
```bash
cd example
flutter run
```

## UI Highlights
- **Typography**: Uses Plus Jakarta Sans for a modern feel.
- **Design System**: Indigo-based theme with Material 3 components.
- **Layout**: Uses `DeskConstraintBox` to keep content centered and readable on large screens.
